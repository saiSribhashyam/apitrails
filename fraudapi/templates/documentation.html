How to Use the API

Use the following endpoint to check a message:

GET http://localhost:8000/api/v1/{apiKey}/check?message={your_message}

Example Request

GET http://localhost:8000/api/v1/YOUR_API_KEY/check?message=Congratulations!%20You%27ve%20won%20a%20free%20vacation.%20Claim%20now%20by%20providing%20your%20personal%20details.

Code for Connecting to the API

Below are examples of how to connect to the API using different programming languages:

JavaScript

fetch('http://localhost:8000/api/v1/YOUR_API_KEY/check?message=YOUR_MESSAGE')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Python

import requests

response = requests.get(
    'http://localhost:8000/api/v1/YOUR_API_KEY/check',
    params={'message': 'YOUR_MESSAGE'}
)
print(response.json())

Java

import java.io.IOException;
import java.net.HttpURLConnection;
import java.net.URL;
import java.util.Scanner;

public class APICall {
    public static void main(String[] args) throws IOException {
        String apiKey = "YOUR_API_KEY";
        String message = "YOUR_MESSAGE";
        URL url = new URL("http://localhost:8000/api/v1/" + apiKey + "/check?message=" + message);
        HttpURLConnection conn = (HttpURLConnection) url.openConnection();
        conn.setRequestMethod("GET");

        int responseCode = conn.getResponseCode();
        if (responseCode == 200) {
            Scanner sc = new Scanner(url.openStream());
            String inline = "";
            while (sc.hasNext()) {
                inline += sc.nextLine();
            }
            System.out.println(inline);
            sc.close();
        } else {
            System.out.println("Error in API Call");
        }
    }
}
